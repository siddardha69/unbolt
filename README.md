# Upbolt Studios — Marketing Website

A high-performance, responsive marketing website for **Upbolt Studios**, a boutique digital marketing and content production agency based in New York. 

Built using Next.js 16 (App Router), React 19, TypeScript, and Tailwind CSS v4.

## 🚀 Key Features

*   **Custom Design System**: Structured with Tailwind CSS v4's CSS-based `@theme` variables, implementing a strict custom color palette and type scale.
*   **Interactive Growth Visualizer**: Interactive client self-qualification tool rendering dynamic DIY vs. Managed trajectory curves via Recharts.
*   **Magnetic Button System**: High-fidelity magnetic hover buttons powered by Framer Motion spring coordinates.
*   **Performance Ticker**: Custom infinite horizontal proof marquee.
*   **Self-Hosted Typography**: Fully local variable fonts using `next/font/local` for fast loading and privacy compliance.
*   **Scroll Animations**: Smooth entrance transitions for page layouts.
*   **Direct Contact Funnels**: Custom Project Intake Form, WhatsApp direct chat link, and booking calendar integrations.

## 🛠️ Tech Stack

*   **Framework**: Next.js 16 (App Router, Strict Mode)
*   **Styles**: Tailwind CSS v4
*   **Animations**: Framer Motion
*   **Data Visualizations**: Recharts
*   **Language**: TypeScript

## 📦 Getting Started

### Prerequisites

Ensure you have Node.js 18.x or later installed.

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

### Production Build

To build and compile the static optimized pages:
```bash
npm run build
```

To run the production bundle locally:
```bash
npm run start
```
